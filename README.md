# Agenda

É uma API REST para inclusão de pessoas e suas listas de contatos.

## Requisitos

* Java 8;
* Maven 3.

## Como executar

Executar o comando abaixo: 

``` bash
mvn spring-boot:run
```

## Documentação

A documentação da API está disponível no [SwaggerHub](https://app.swaggerhub.com/apis/thaisalonso/agenda/1.0).
